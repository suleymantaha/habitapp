# Settings

## Amaç
Hesap ve uygulama tercihlerini yönetmek.

## Bölümler
- Account
  - Restore purchases
  - Manage subscription
- Defaults
  - Currency
  - WhatsApp number
  - Message template
- App
  - Language (opsiyonel)
  - Rate the app

## Başarı kriteri
- Kullanıcı 2 tıkla aboneliğini yönetebilir.
