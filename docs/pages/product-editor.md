# Product Editor

## Amaç
Tek bir ürün/hizmetin detaylarını yönetmek.

## Alanlar
- title (zorunlu)
- price (zorunlu)
- description (opsiyonel)
- photo (opsiyonel)
- categoryTag (opsiyonel)

## Eylemler
- Save
- Delete
- Duplicate (opsiyonel)

## Validasyon
- title: 2–60 karakter
- price: 0’dan büyük (ücretsiz ürün için 0 desteklenebilir, ürün stratejisine göre)

## Fotoğraf akışı
- Gallery seçimi
- Crop (opsiyonel)
- Sıkıştırma: paylaşım çıktıları için 1080 genişlik hedeflenir
