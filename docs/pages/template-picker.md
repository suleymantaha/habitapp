# Template Picker

## Amaç
Kataloğun görsel dilini seçmek ve çıktı kalitesini yükseltmek.

## Ekran öğeleri
- Template grid (önizleme kartları)
- Filtre: “Food”, “Beauty”, “Boutique”, “Minimal”
- Toggle: “Show watermark”

## Davranış
- Seçilen template anında preview eder
- Seçim kaydedilince Export & Share’a dönüş

## Premium kurgusu
- Bazı template’ler kilitli (Premium)
- Kilitliye tıklayınca Paywall açılır
